## Introduction

We built this site to serve as our online home base.

## Tech Stack

The site is built in React on top of Next.js, and hosted on Vercel. A few important dependencies include [Shadcn UI](https://ui.shadcn.com) and [contentlayer](https://www.contentlayer.dev/). Feel free to check out the code for more details.
